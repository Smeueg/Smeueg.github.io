My Very Smeueg Dotfiles

These dotfiles consists of configuration files for: AwesomeWM, Emacs, rofi (optional, AwesomeWM also has it's own way to start programs ), and any posix shells

To Bootstrap my dotfiles, if you're using something like bash or zsh run:

sh <(curl -L https://smeueg.github.io/smeueger)

or

sh <(wget -O- https://smeueg.github.io/smeueger)


else, run:

sh <<EOF $(curl -L https://smeueg.github.io/smeueger) EOF

or

sh <<EOF $(wget -O- https://smeueg.github.io/smeueger) EOF





I also have a youtube playlist downloader which downloads every song in a playlist to the mp3 format:

curl -Ls https://smeueg.github.io/smeuesic -o /tmp/smeuesic; sh /tmp/smeuesic

or

wget https://smeueg.github.io/smeuesic -qO /tmp/smeuesic; sh /tmp/smeuesic
