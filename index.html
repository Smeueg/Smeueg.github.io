Welcome fleshlings! You'll find things like my dotfiles and some of my scripts or programs that I'd like to showcase. I hope you enjoy your stay!

These dotfiles has configuration files for:

  • AwesomeWM
  • Emacs
  • Rofi (Optional, as AwesomeWM also has it's own "menu")
  • Any POSIX shell like bash


I made a script to easily bootstrap my dotfiles. You should be able to set it up with this one liner using the terminal/TTY:

curl -L https://smeueg.github.io/smeueger -o /tmp/smeueger && sh /tmp/smeueger

Or if don't have curl installed but instead have wget:

wget https://smeueg.github.io/smeueger -O /tmp/smeueger && sh /tmp/smeueger
curl -L https://smeueg.github.io/smeuesic -o /tmp/smeuesic && sh /tmp/smeuesic

wget https://smeueg.github.io/smeuesic -O /tmp/smeuesic && sh /tmp/smeuesic
